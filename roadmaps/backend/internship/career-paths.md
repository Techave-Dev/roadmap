## Career Progression Paths

### Path 1: Individual Contributor (IC) Track

```
Junior Backend Engineer (0-2 years)
└─→ Backend Engineer (2-4 years)
    └─→ Senior Backend Engineer (4-7 years)
        └─→ Staff Engineer (7-10 years)
            └─→ Principal Engineer (10+ years)
```

#### Junior Backend Engineer (Entry)

**Responsibilities:**
- Implement well-defined features
- Write tests
- Participate in code reviews (receiving)
- Learn codebase and team practices

**Skills Needed:**
- ✅ Competent with one backend framework (NestJS ✓)
- ✅ Understands databases and ORMs (Prisma ✓)
- ✅ Can write tests (Unit + Integration ✓)
- ✅ Solid Git workflow ✓

**Salary Range (US):**
- Tech hubs: $80k-$110k
- Other cities: $65k-$85k
- Remote: $70k-$100k

**Growth Timeline:** 18-24 months to next level

---

#### Backend Engineer (Mid-Level)

**Responsibilities:**
- Own features end-to-end
- Design solutions
- Mentor junior engineers
- Participate in architectural discussions
- Give code reviews

**Skills Needed:**
- ✅ Proficient in backend framework ✓
- ✅ Database optimization ✓
- ✅ System design basics ✓
- ✅ Production debugging ✓
- ✅ CI/CD experience ✓

**Salary Range (US):**
- Tech hubs: $110k-$150k
- Other cities: $85k-$120k
- Remote: $90k-$130k

**Growth Timeline:** 2-3 years to next level

---

#### Senior Backend Engineer

**Responsibilities:**
- Architect systems
- Lead projects
- Mentor team members
- Make technical decisions
- Define standards and best practices
- On-call rotation (production support)

**Skills Needed:**
- Expert in backend development
- Advanced system design
- Performance optimization
- Security hardening
- Cross-functional collaboration

**Salary Range (US):**
- Tech hubs: $150k-$220k + equity
- Other cities: $120k-$180k + equity
- Remote: $130k-$200k + equity

**Growth Timeline:** 3-4 years to Staff level

---

### Path 2: Management Track

```
Junior Backend Engineer
└─→ Backend Engineer
    └─→ Senior Backend Engineer
        └─→ Tech Lead (hybrid role)
            └─→ Engineering Manager
                └─→ Senior Engineering Manager
                    └─→ Director of Engineering
                        └─→ VP of Engineering
```

#### Tech Lead (4-6 years experience)

**Transition Point:** Still coding 50-70%, but also:
- Leading team of 3-5 engineers
- Planning sprints
- Unblocking team members
- Technical decision-making
- Stakeholder communication

**Salary:** Similar to Senior Engineer + leadership premium

---

#### Engineering Manager (6-8 years)

**Responsibilities:**
- Manage team of 5-8 engineers
- Hiring and performance reviews
- Career development
- Limited coding (10-30%)
- Project planning and delivery

**Salary Range (US):**
- $150k-$250k + equity + bonus

---

### Path 3: Specialist Tracks

#### Backend Architect

**Focus:** System design, architecture, scalability
**Timeline:** 7-10 years
**Salary:** $180k-$300k+

---

#### DevOps / Platform Engineer

**Transition:** Learn Kubernetes, infrastructure
**Timeline:** From Backend Engineer role
**Salary:** Similar to Backend Engineer

---

#### Security Engineer

**Transition:** Deep dive into security
**Timeline:** 3-5 years backend + security focus
**Salary:** $130k-$250k+

## Company Types & Career Impact

### Startups (Early Stage)

**Pros:**
- Wear many hats (rapid learning)
- High impact (small team)
- Equity potential (if successful)
- Fast-paced environment

**Cons:**
- Lower salary
- Less mentorship
- Higher risk
- Longer hours

**Best For:** After 1-2 years experience, want rapid growth

---

### Startups (Late Stage / Unicorns)

**Pros:**
- Startup energy, better resources
- Competitive salary + equity
- Established processes
- Growth opportunities

**Cons:**
- Less ownership than early stage
- More politics than small startups

**Best For:** Mid-level engineers wanting balance

---

### Big Tech (FAANG+)

**Pros:**
- Highest salaries
- Best benefits
- Name recognition on resume
- Cutting-edge technology
- Strong mentorship

**Cons:**
- Competitive (hard to get in)
- Slower pace
- More bureaucracy
- Specialized roles

**Best For:** Mid-Senior engineers wanting stability + compensation

---

### Enterprise

**Pros:**
- Stable
- Good work-life balance
- Structured career paths

**Cons:**
- Legacy technology (sometimes)
- Slower innovation
- Lower compensation than tech companies

**Best For:** Value stability over cutting-edge

## Final Thoughts

**Your career is a marathon, not a sprint.**

This internship gives you a strong foundation. Where you go from here depends on:
- Continuous learning
- Building relationships
- Delivering impact
- Finding what excites you

**Everyone's path is different.** Some reach Senior in 4 years. Others take 7. Both are valid.

**Focus on growth, not comparison.**
